## About p5.js

p5.js is a JavaScript library for creative coding, with a focus on making coding accessible and inclusive for artists, designers, educators, beginners, and anyone else! It is a free and open-source library with a friendly community and comprehensive documentation.

**Learn more:** [https://p5js.org/](https://p5js.org/)

### Key Features
- Easy to learn and use
- Built-in drawing and animation functions
- Support for interactive graphics
- Sound and media capabilities
- Active community and extensive examples

## Getting Started

Each project folder contains:
- `index.html` - The main HTML file
- `sketch.js` - The p5.js sketch code
- `style.css` - Styling
- `p5.js` - The p5.js library
- Individual project README with specific details

To run a project click the "Run here" links above to try them in the p5.js web editor
